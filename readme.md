# Inbox app

## Prerequisites

- Install MongoDB
- Install Nodejs

## Run steps

- Install node_modules of the server

`npm install`

- Add mock messages to the database

`node addMessages.js`

- run the server

`npm start`

- Install node_modules of the client

`cd client/`
`npm install`

- run the client

`npm start`

- Go to:

`localhost:3000/`
